# [BOJ_거스름돈_5585](https://www.acmicpc.net/problem/5585)
***
### My_code
```python
price = int(input())

change = 1000 - price
count = 0
while change != 0:
    while change >= 500:
        change -= 500
        count += 1
    while change >= 100:
        change -= 100
        count += 1
    while change >= 50:
        change -= 50
        count += 1
    while change >= 10:
        change -= 10
        count += 1
    while change >= 5:
        change -= 5
        count += 1
    while change >= 1:
        change -= 1
        count += 1

print(count)
```
